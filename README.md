# my_g102_cs_click
用 rust 語言開發，用來取代 GHub 讓側鍵可以連點打 CS 比較方便的外掛

▶️ 使用方式

1️. 安裝 Rust

https://rustup.rs

2. 下載或 clone 此專案
git clone https://github.com/shadowjohn/my_g102_cs_click

3️. 建置 release exe
cd my_g102_cs_click
cargo build --release

4️. 
執行 target/release/my_g102_cs_click.exe
按住 G102 左後側鍵（XBUTTON1）即可連點觸發左鍵 10ms 速度開槍
